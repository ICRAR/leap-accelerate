# LEAP Accelerate

Acceleration module for LEAP-cal for GPU accelerated phase rotation.

## Install

### Linux

`mkdir build && cd build`
`mdir linux && cd linux`
`cmake ../../`

## Testing

Testing provided via googletest.

in build directory run:
`make test`

## Run Server

`./build/linux/src/icrar/leap-accelerate/`
